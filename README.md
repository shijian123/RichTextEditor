# 图文编辑器

目前编辑器支持图文混排，并可以插入表情包、图片、外部链接卡片。

此编辑器由HTML构建，编辑完成后对HTML标签解析生成内容数组，可将此数组传递给服务器，最后根据内容数组生成展示富文本页面
